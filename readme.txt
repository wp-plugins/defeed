Create custom RSS feeds that will fit your requirements.

== Description ==
De:feed help you create RSS feed, in user-friendly drag-n-drop interface, according to the requirements that you have for it.

Currently it can use posts, any custom fields that used for posts, and any taxonomies both inbuilt like tags or custom, please check screenshots section for more clarity.

We\'ll be glad to meet you in support section, with any thoughts about subject, with feature requests or bug reports.

Have a great feeding :)

== Installation ==
1. Download plugin through the \"Plugins\" menu in WordPress.
1. Activate.
1. Navigate to the \"De:feed\" menu, and create your first feed.
